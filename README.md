# 体調管理アプリ

自分の体調・食事・行動を記録して、不調の原因を統計的に分析するスマートフォンアプリです。

> **このアプリは [Claude Code](https://claude.ai/code) を使って開発しました。**

---

## 機能

### 記録
- **体調記録** — 1日に複数回、時間帯ごとにスコア・症状・メモを記録
- **行動・食事記録** — カスタムカテゴリで自由に記録（食事、アルコール、睡眠、運動など）
- **過去日入力対応** — 忘れた日の記録も遡って入力できる
- **時間帯選択** — 朝・午前・昼・午後・夕方・夜・深夜の7区分でざっくり入力

### カスタムカテゴリ
- 自由記述・選択式（ラジオ/プルダウン）を自分で設定
- デフォルトカテゴリ（食事・アルコール・睡眠・運動・カフェイン・ストレス）付き
- 選択肢は1行1つで自由に追加

### 分析
- **ラグ相関分析** — 最大14日前の行動・食事と体調スコアの相関を自動計算
- **累積効果検出** — 「3日連続アルコール → 不調」のようなパターンも検出
- 5件以上の記録が溜まると分析が有効になる

### セキュリティ
- **生体認証** — Face ID / 指紋認証でアプリをロック
- **暗号化ストレージ** — データはiOS Keychain / Android Keystoreに暗号化して保存
- インターネット接続不要・完全ローカル動作

## セットアップ

### 必要なもの
- Node.js 18以上
- スマートフォン（[Expo Go](https://expo.dev/go) アプリをインストール）

### 手順

```bash
# リポジトリをクローン
git clone https://github.com/halso0218/health-tracker.git
cd health-tracker

# 依存関係をインストール
npm install

# 開発サーバーを起動
npx expo start
```

表示されたQRコードをスマートフォンのカメラで読み込むと起動します。

## 画面構成

```
ホーム      — 今日の体調サマリーと記録件数
体調タブ    — 体調スコア・症状を時間帯つきで記録
記録タブ    — 食事・行動などをカテゴリ別に記録
分析タブ    — 不調の原因を最大14日前のデータから分析
```
